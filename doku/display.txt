Kommunikation + Handshake:
--------------------------
ATmega → ESP32:
═══════════════════════════════════════════════════════
Schritt │ ATmega (WR)          │ ESP32 (RD)           │
────────┼──────────────────────┼──────────────────────┼
  1     │ WR=HIGH (idle)       │ RD=HIGH (idle)       │
  2     │ schreibt auf Bus     │                      │
  3     │ WR=LOW               │ sieht WR=LOW         │
  4     │                      │ RD=LOW (ACK)         │
  5     │ wartet auf RD=LOW    │                      │
  6     │ WR=HIGH (gültig)     │ warte auf WR=HIGH    │
  7     │                      │ liest Bus            │
  8     │                      │ RD=HIGH (idle)       │
  9     │ warte auf RD=HIGH    │                      │
  10    │ WR=HIGH (idle)       │                      │ 
═══════════════════════════════════════════════════════

ESP32 → ATmega:
═══════════════════════════════════════════════════════
Schritt │ ESP32 (RD)           │ ATmega (WR)          │
────────┼──────────────────────┼──────────────────────┼
  1     │ RD=HIGH (idle)       │ WR=HIGH (idle)       │
  2     │ schreibt auf Bus     │                      │
  3     │ RD=LOW               │ sieht RD=LOW         │
  4     │                      │ WR=LOW (ACK)         │
  5     │ warte auf WR=LOW     │                      │
  6     │ RD=HIGH (gültig)     │ warte auf RD=HIGH    │
  7     │                      │ liest Bus            │
  8     │                      │ WR=HIGH (idle)       │
  9     │ warte auf WR=HIGH    │                      │
  10    │ RD=HIGH (idle)       │                      │
═══════════════════════════════════════════════════════
